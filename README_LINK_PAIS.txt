LINK DOS PAIS

Depois de publicar no Vercel, envie para os pais o link com ?pais=1 no final.

Exemplo:
https://SEU-SITE.vercel.app?pais=1

Esse modo mostra somente:
- Ranking geral do mês
- Abas por categoria com resultados
- Logo Primo Soccer

Não mostra cadastro, agenda, disputa, config ou edição.
