Primo Soccer League - versão com reset.html

Versão: 2.0.1

Novo recurso:
- reset.html para limpar Service Worker e caches antigos do iPhone.

Como usar:
1. Publique este ZIP no Netlify.
2. Abra: https://SEU-LINK.netlify.app/reset.html
3. Toque em "Limpar cache e atualizar".
4. O app abrirá com a versão nova.
5. Depois adicione novamente à tela inicial, se necessário.

Também existe botão dentro do app:
Config. > Reset total iPhone
