PRIMO SOCCER LEAGUE - VERSÃO PARA VERCEL

Como publicar no Vercel:

1. Acesse https://vercel.com
2. Faça login.
3. Clique em Add New Project.
4. Envie os arquivos deste ZIP descompactado ou importe pelo GitHub.
5. Framework Preset: Other.
6. Build Command: deixe vazio.
7. Output Directory: deixe vazio ou coloque .
8. Clique em Deploy.

Importante:
- Este app continua conectado ao Supabase.
- Os dados dos atletas ficam salvos no banco online.
- O link dos alunos fica:
  https://SEU-PROJETO.vercel.app/?aluno=1

Arquivos principais:
- index.html
- style.css
- script.js
- supabase-config.js
- sw.js
- vercel.json
